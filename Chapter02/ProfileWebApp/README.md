# Profile Web Application

A dual-frontend profile webpage with Angular and Blazor WebAssembly, featuring toggle buttons to switch between the two frameworks.

## Running the Applications

### Blazor (C# .NET) App
```bash
cd ProfileWebApp
dotnet run
```
The Blazor app will run on `http://localhost:5000`

### Angular App
```bash
cd ProfileWebApp/AngularClient
ng serve
```
The Angular app will run on `http://localhost:4200`

## Features
- Toggle buttons to switch between Angular and Blazor frontends
- "Hello, world!" header on both applications
- Clean, minimal design ready for expansion
